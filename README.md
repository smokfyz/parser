# Инструкция

Тестирование проводилось на:
Python 3.8.5
MacOS 11.1

1. ```python -m venv env```
2. ```. ./env/bin/activate```
3. ```pip install -r requiremets.txt```
4. Установить graphviz. Для brew: ```brew install graphviz```
5. Написать код в input.txt
6. ```python parser.py```
7. Абстрактное синтаксическое дерево находится в файле input.png
